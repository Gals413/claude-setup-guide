Section 1

What Is Claude? (And Why You Need It)

Claude is an AI assistant made by Anthropic. Think of it as a colleague who can read documents, analyze data, write reports, and answer questions -- in plain English. No formulas. No macros. Just tell it what you need.

Claude comes in three forms:

⚭

claude.ai

Web browser. Quick access. Cannot connect to your systems.

■

Claude Desktop

App on your computer. Can connect to systems via MCP.

▶

Claude Code

Terminal-based. Most powerful. Automation, scripting, system connections.

ⓘ
Why does this matter for finance? Claude Desktop and Claude Code can connect to NetSuite, Gmail, Google Calendar, and other systems using a protocol called MCP (Model Context Protocol). The web version at claude.ai cannot. If you want to query your ERP from Claude, you need one of the installed versions.

Recommendation: Start with Claude Desktop. It works immediately, feels familiar, and supports MCP connections to NetSuite and other systems. If you later want automation or advanced workflows, add Claude Code.

Section 5 -- Step 3

Install Claude Code (Optional but Powerful)

Claude Code runs in your terminal -- the text-based interface on your computer. It is more powerful than Claude Desktop because it can directly access your file system, run scripts, and automate multi-step workflows.

When do you need it?

  • ✓ Connecting to enterprise systems with advanced configuration
  • ✓ Automating repetitive tasks (reports, data processing, file management)
  • ✓ Building custom tools and workflows
  • ✓ Working with code, scripts, or configuration files
ⓘ
If the terminal feels intimidating, skip this section for now. Claude Desktop handles most use cases, including MCP connections to NetSuite. You can always come back and install Claude Code later.
3
Install Claude Code

What is Terminal? Terminal is a built-in app on every Mac. It lets you type commands instead of clicking buttons. You will only need to type a few lines -- we will tell you exactly what.

  1. 1
    Open Terminal. Press Cmd + Space to open Spotlight, type Terminal, and hit Enter. A window with a blinking cursor will appear.
  2. 2
    Install Claude Code. Copy and paste the command below into Terminal, then press Enter.
Terminal -- Mac
# Install Claude Code
~ $ npm install -g @anthropic-ai/claude-code
ⓘ
If you see "npm: command not found" -- you need to install Node.js first. Go to nodejs.org, download the LTS version, install it, then close and reopen Terminal before retrying the command above.
  1. 3
    Close Terminal and open a new window. This refreshes your system path so it recognizes the claude command.
  2. 4
    Type claude and hit Enter.
  3. 5
    Your browser will open for authentication. Sign in with the same account you used for Claude Desktop. Once authenticated, return to Terminal -- you are in.
Terminal -- Mac
~ $ claude
Opening browser for authentication...
Authentication successful.
Welcome to Claude Code. Type your question or task.

What is PowerShell? PowerShell is a built-in tool on every Windows PC. It lets you type commands. You will only need a few lines -- we will give you exactly what to type.

  1. 1
    Install Node.js first. Go to nodejs.org, download the LTS version, and run the installer. Accept all defaults.
  2. 2
    Open PowerShell. Press Win + X and select "Windows PowerShell" (or "Terminal") from the menu. A blue/black window with a cursor will appear.
  3. 3
    Install Claude Code. Copy and paste the command below into PowerShell, then press Enter.
PowerShell -- Windows
# Install Claude Code
PS C:\> npm install -g @anthropic-ai/claude-code
⚠
Use PowerShell, not Command Prompt (CMD). If you see C:\> with no "PS" prefix, you are in CMD. Close it and open PowerShell instead. CMD does not support the required commands.
  1. 4

  2. 5
    Type claude and hit Enter.
  3. 6
    Your browser will open for authentication. Sign in with the same account you used for Claude Desktop. Once authenticated, return to PowerShell -- you are in.

Troubleshooting

Common Issues and Fixes

Click any issue to expand the fix.

"command not found: claude"
Your terminal does not know where Claude Code is installed. Close the terminal window entirely and open a fresh one. If it still does not work, the npm global bin directory may not be in your PATH. Run npm bin -g to find the directory, then add it to your shell profile (~/.zshrc on Mac, or your System Environment Variables on Windows).
"npm is not recognized" or "irm is not recognized"
For "npm": You need to install Node.js first. Download it from nodejs.org (LTS version), run the installer, then close and reopen your terminal.

For "irm": You are in Command Prompt (CMD), not PowerShell. Close CMD and open PowerShell instead. Press Win + X and select "Windows PowerShell".
Browser does not open for authentication
The terminal should display a URL. Copy it manually and paste it into your browser. Complete the sign-in there, then return to the terminal. It should detect the authentication automatically.
"Permission denied" or "EACCES" error
Do NOT use sudo -- it creates ownership problems. Instead, fix the npm global directory permissions. On Mac, run: sudo chown -R $(whoami) $(npm config get prefix)/{lib/node_modules,bin,share} then retry the install without sudo. On Windows, try running PowerShell as Administrator (right-click, "Run as administrator").
403 Forbidden or subscription error
Check that your Claude subscription is active at claude.ai/settings. Claude Code requires an active Pro, Max, or Teams plan. If you just subscribed, sign out and sign back in to refresh your session.
